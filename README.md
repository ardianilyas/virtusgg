# Virtus

Virtus is a task management web app build using Laravel Inertia Vue

### Tech Stack :
- Laravel
- Inertia
- Vuejs
- Tailwindcss
- Shadcn vue
- Redis
- Mailtrap
- Reverb

### Features (current) : 
- Sign In
- Sign Up
- Reset password
- Email verification
- Manage organization
- Realtime notification request join using reverb
- Request to join organization
- Accept users request
